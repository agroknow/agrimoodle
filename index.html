Welcome to agriMoodle

The concept...

agriMoodle is a feature-rich, fresh Moodle 2.x installation bundle, that:

  • provides great green/agro- templates and themes;
  • is linked to interesting green/agro- information sources;
  • trainers can use tools to import resources from relevant repositories;
  • trainers can use an easy LOM editor to describe and export metadata for their resources;
  • learners can benefit from (automatic) references to useful material, through various widgets.

Read more about the agriMoodle concept and vision.
