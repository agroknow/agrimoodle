NOTES
=====
*INSTALLATION
 - Make sure you have all the required versions. 
	- moodle download from here: https://github.com/agroknow/agrimoodle/tree/with_marinos, 
	- just block oai-target download from here: https://github.com/agroknow/agrimoodle/tree/with_marinos/blocks/oai_target
 - Download and unpack the block folder.
 - Place the folder (eg "myblock") in the "blocks" subdirectory.
 - Login in your Moodle platform as Administrator and
   click on OAI Target inside Site Administration block.
   At this point the block initialization should be performed and the block
   should be available in the Blocks list. Note that the block is available
   only inside courses.
 - Turn editing on in any course page.
 - Add the block to the page.
 - Use Metadata Editor to fill in the Metadata (See Metadata Edidor Notes).
 - Run Cron through your moodle.
 - You are ready to use OAI-TARGET.
